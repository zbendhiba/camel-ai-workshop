Welcome

A hands-on workshop to learn how to build intelligent data pipelines using Apache Camel, Forage, and Large Language Models.

Unlike traditional AI workshops that focus on chatbot interactions, this workshop demonstrates how Camel's Enterprise Integration Patterns — Content-Based Router, Splitter, Aggregator, Dynamic Router — combine with LLM capabilities to create production-grade AI data pipelines.

Two approaches to LLM integration

Each step is available in two variants:

  • OpenAI component (openai:chat-completion) — the simplest way to call an LLM from Camel. Works with OpenAI or any OpenAI-compatible API like Ollama.
  • LangChain4j-agent (langchain4j-agent) via Forage — adds memory, tool support, and guardrails. Shines for advanced use cases.
